A beautiful, mobile-first birthday countdown website with a soft baby pink and light crayon purple theme.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Add your assets to `src/assets/`:
   - `her1.png` (primary image)
   - `her2.png` (supporting image)
   - `her3.png` (supporting image)
   - `video.mp4` (9:16 vertical video)

3. Run the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Configuration

### Birthday Date
Edit the `BIRTHDAY_DATE` constant in `src/App.jsx` to change the birthday date. Currently set to January 2nd, IST.

### Messages
Edit the `MESSAGES` array in `src/components/Messages.jsx` to customize the sweet messages for Urvashi.
